# Job Tracker Web Application

A full-stack web application designed to help users track, manage, and analyze job applications in one centralized platform.

## 🚀 Live Demo
https://job-tracker-sigma-gules.vercel.app/

## 💻 GitHub Repository
https://github.com/gurkamaldhiman3002-create/job-tracker

## 🧠 Features
- Add, update, and delete job applications
- Track application status (Applied, Interview, Rejected)
- Filter, search, and sort applications
- Source tracking (LinkedIn, Indeed, etc.)
- Analytics dashboard (timeline + source insights)
- Secure user authentication for personalized data

## 📊 Metrics
- Tracked 25+ job applications using the platform
- Built analytics to monitor application trends and interview conversion
- Identified top-performing job sources based on outcomes

## 🛠 Tech Stack
- Frontend: Next.js
- Backend: Supabase (Database + Authentication)
- Deployment: Vercel
- Version Control: GitHub

## ⚙️ Installation
git clone https://github.com/gurkamaldhiman3002-create/job-tracker
cd job-tracker
npm install
npm run dev
